- This emulator uses 0.2.96.86 FBA romset
- Standard keymapping:
```
D-PAD        - D-PAD
SELECT       - Coin1
START        - Start1
SELECT+START - Start2
A            - Fire1
B            - Fire2
X            - Fire3
Y            - Fire4
L            - Fire5
R            - Fire6

L+R+Y        - Show/hide fps
L+R+A        - Quick state load
L+R+B        - Quick state save
L+R+SELECT   - Service menu
L+R+START    - FBA menu (key config and load/save states)
```
